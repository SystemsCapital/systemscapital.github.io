S&P 500 Company Data

This table presents detailed short-term qualities for each stock included in the S&P 500 index, and is updated daily. The data below provides valuable insights into daily stock market performance. Using Python libraries such as Beautiful Soup 4 and Requests, we programmatically collect an up-to-date list of S&P 500 companies from Wikipedia. Daily price and earnings data in recent days is then retrieved using Microsoft Excel’s STOCKHISTORY and LET functions. Then a JSON is created using our Excel-to-JSON tool (found: here) and the data is displayed on our table below. All information is provided on an “as-is” basis and reflects price-based calculations only; dividend data is not included.
